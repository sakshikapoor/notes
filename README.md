# NOTES

Note taking app, made with React/Redux.

## Features

1. Notes can be pinned, archived, or trashed
2. Search (looks for exact substring in not title or content)

4. State is persisted to local storage

## Code Structure

- All code is in `/src`
- Plain CSS used for styling.
- React componenet are in `/src/components`
- Redux actions, reducers in directories named accordingly

## How to run

1. Clone the repo
2. cd into the cloned directory
3. `npm install` to install dependencies
4. `npm start`
5. Visit localhost:3000 to see the app in action
